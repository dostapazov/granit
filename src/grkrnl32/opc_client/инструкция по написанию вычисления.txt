Скрипт вычисленя значение запускается 
со следующими глобальными переменными:
item_id       - строка Tag name 
item_value    - Значение 
item_quality  - Качество
Доступны для вызова процедуры

__trace (string text)  
функция выводит текст в монитор исполнения скриптов

Получить значение тега
get_value(string arg_item_id) 
 возвращает 3 значения 
 1) 0 - нет переменной 1 - ok 
 2) текущее значение переменной
 3) качество переменной

Установить значение тега
set_value(value number , tag_name string )

Установить диагностику по линии 
set_line_diag(int diag)
Нет связи с КП             0x08000000
КП не отвечает             0x04000000


Установить качество тегов по ФА-ГРУППЕ
set_group_quality(int quality,int fa = -1,int group =-1 );
fa    - значение по умолчанию -1 т.е по всем фа
group - значение по умолчанию -1 т.е по всем группам.
Для справки качество = 192 - OK
		     = 8   - нет связи
		     = 12  - неисправность устройства

int bin_and(int v1,int v2  ) - возвращает результат двоичного 
И
int bin_or (int v1,int v2  ) - возвращает результат двоичного 
ИЛИ
int bin_xor(int v1,int v2  ) - возвращает результат двоичного 
имключающего ИЛИ
int bin_not(int v          ) - возвращает инверсное значение 
аргумента
int bin_shl(int v1,int n   ) - возвращает результат сдвига 
влево  на n бит
int bin_shr(int v1,int n   ) - возвращает результат сдвига 
вправо на n бит
