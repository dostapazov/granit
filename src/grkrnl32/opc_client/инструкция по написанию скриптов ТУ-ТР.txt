Скритп запускается со следующими глобальными переменными
Константы значений команд
1) cmd_on_more      ВКЛ-БОЛЬШЕ
2) cmd_off_less     ОТКЛ-МЕНЬШЕ
3) cmd_tr_set       Установка значения  
4) cmd_undo         Отмена опреации


Переменные 
9)  item_id      - id  OPC переменной над которой проводится опреация ТУ/ТР
10) rc_object    - адрес ту соответствующй объекту
11  rc_tag       -  имя тега  для команд ТУ
12) rc_command   - код операции см. cmd_XXX  пп.1-4 
13) rc_value     - для ТР Значение до которого осуществлять регулирование

  ** Доступны следующие внешние функции **
sleep( integer N ) Приостановка выполнения на   N миллисекунд

check_terminate () Проверка необходимости завершения  
возвращает не 0 при требовании завершения
необходимо отправить отмену опрерации

int bin_and(int v1,int v2  ) - возвращает результат двоичного И
int bin_or (int v1,int v2  ) - возвращает результат двоичного ИЛИ
int bin_xor(int v1,int v2  ) - возвращает результат двоичного имключающего ИЛИ
int bin_not(int v          ) - возвращает инверсное значение аргумента
int bin_shl(int v1,int n   ) - возвращает результат сдвига влево  на n бит
int bin_shr(int v1,int n   ) - возвращает результат сдвига вправо на n бит



Для всех нижеследующих функций параметр arg_item_id - необязательный. 
В этом случае arg_item_id берется  из глобальной переменной item_id;

set_rc_prepare(int set  ,string arg_item_id)   см. установка сброс подготовки
set_rc_active (int set  ,string arg_item_id)   см. установка сброс активности
set_rc_error  (int set  ,string arg_item_id)   см. установка сброс ошибки

get_value(string arg_item_id) 
 возвращает 3 значения 
 1) 0 - нет переменной 1 - ok 
 2) текущее значение переменной
 3) качество переменной

set_value(double value,string arg_item_id) Запись значения opc
